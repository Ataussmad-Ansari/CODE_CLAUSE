Data types in C

Data in C specifies the type of data that the variable can store like integer, character, floating, double, etc. Each data type requires different amounts of memory and has some specific operations which can be performed over it. The data type is a collection of data with values having fixed values, meaning as well as its characteristics.

TypesDiscription
Primitive Data TypesPrimitive data types are the most basic data types that are used for representing simple values such as integers, characters, floating points, double floating points and Void
User-defined Data TypesThe user-defined data types are defined by the user himself. Class, Structure, Union, Enum and Typedef are the user-defined data types.
Derived Data TypesThe data types that are derived from the primitive or built-in datatypes are referred to as Derived Data Types. Function, Array, Pointer and Reference are the user-defined data types.

Basic Data Types:

  • int: Used to store integer values.
  • char: Used to store a single character.
  • float: Used to store single-precision floating-point numbers.
  • double: Used to store double-precision floating-point numbers.

Derived Data Types:

  • Array: A collection of elements of the same data type.
  • Pointer: A variable that stores the memory address of another variable.
  • Structure: A user-defined data type that groups related variables together.
  • Union: Similar to a structure but allows variables to share the same memory location.
  • Enum: A user-defined data type that assigns names to integer values.

Void Data Type:

  • void: Used to indicate that a function does not return any value or that a pointer does not have a specific data type.

Modifiers:

  • short: Modifies an integer to make it shorter.
  • long: Modifies an integer or double to make it longer.
  • signed and unsigned: Modifiers used with integers to specify whether they can hold both positive and negative values or just positive values.

Typedef:

  • typedef: Used to create user-defined data type names for existing data types, making code more readable and maintainable.

*A format specifier in C is a special character or sequence of characters used to indicate the kind and format of data that will be input into or produced from the program.
