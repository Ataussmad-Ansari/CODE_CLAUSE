printf(), scanf() in C

In the C programming language, printf and scanf are two important functions that are used for input and output operations. They are part of the standard input/output library in C.

printf:

  • The printf() function is used for output (displaying text) in the C programming language.
  • It allows you to format and print text to the standard output (usually the console).
The general syntax is:
printf("format_string", arg1, arg2, ...);
  • The "format_string" is a string containing format specifiers that control how the arguments are formatted and displayed. For example, %d is used to display an integer, %f for a floating-point number, and %s for a string.
  • The arg1, arg2, etc., are the values that will be substituted into the format specifiers in the order they appear in the printf statement.
int num = 9;
printf("Value of num = %d", num);

scanf:

  • scanf is used for input (reading data) in the C programming language.
  • It allows you to read values from the standard input (usually the keyboard) and store them in variables.
The general syntax is:
scanf("format_string", &var1, &var2, ...);
  • The "format_string" specifies the expected format of the input, just like in printf. You use format specifiers to indicate the type of data you expect.
  • The '&' symbol is used to get the memory address of the variables where the input values should be stored.
int num;
printf("%d", &num);

It's important to be careful when using scanf to ensure that the input matches the format specified in the format string, as incorrect input can lead to unexpected behavior or errors. Additionally, input validation is often required to handle cases where the user provides unexpected or incorrect input.

It's important to note that printf and scanf are powerful but can be prone to errors if not used carefully. Mistakes like format specifier mismatches, buffer overflows, or not handling user input errors can lead to unexpected behavior in your C programs. Therefore, it's essential to use these functions with caution and follow best practices in C programming.
