Array in C

An array is defined as the collection of similar type of data items stored at contiguous memory locations. Arrays are the derived data type in C programming language which can store the primitive type of data such as int, char, double, float, etc. It also has the capability to store the collection of derived data types, such as pointers, structure, etc. The array is the simplest data structure where each data element can be randomly accessed by using its index number.

Declaration of C Array

We can declare an array in the c language in the following way

data_type array_name[array_size];

Now, let us see the example to declare the array.

int marks[5];

Here, int is the data_type, marks are the array_name, and 5 is the array_size.

Initialization of Array

The simplest way to initialize an array is by using the index of each element. We can initialize each element of the array by using the index. Consider the following example.

marks[0] = 58; //Initialization of array
marks[1] = 80;
marks[2] = 60;
marks[3] = 90;
marks[4] = 86;
58 marks[0]

Initialization of Array

C array example

#include<stdio.h>
int main() {
    int i = 0;
    int marks[5]; //declaration of array
    marks[0] = 58; //Initialization of array
    marks[1] = 80;
    marks[2] = 60;
    marks[3] = 90;
    marks[4] = 86;
    //traversal of array
    for(i = 0; i < 5; i++){
        printf("%d \n", marks[i]);
    } //end for loop
    return 0;
}

Declaration with Initialization

We can initialize the c array at the time of declaration. Let's see the code. 

int name[5] = {20, 40, 32, 54, 67};

In such case, there is no requirement to define the size. So it may also be written as the following code. 

int name[] = {20, 40, 34, 60, 23;}

Let's see the C program to declare and initialize the array in C.


#include <stdio.h>

int main() {
    int myArray[5] = {10, 20, 30, 40, 50}; //declaration and Initialization
    
    // Access and print array elements
    for (int i = 0; i < 5; i++) {
        printf("Element %d: %d\n", i, myArray[i]);
    }
    
    return 0;
}

Output

10
20
30
40
50

Advantages:

1. Efficient Access: Array elements are stored in contiguous memory locations, allowing for quick and efficient random access using indices. This makes arrays ideal for scenarios where you need to access elements by their position.

2. Memory Efficiency: Arrays are memory-efficient because they allocate a fixed amount of memory for a specific number of elements. This can be more efficient than dynamic data structures if you know the exact number of elements you need in advance

3. Simplicity: Arrays are straightforward to use and understand. They are a fundamental concept in C, and basic operations like element access and modification are easy to perform.

Disadvantages:

1. Fixed Size: Arrays in C have a fixed size, and this size must be known at compile time. This limitation can be problematic if you need to store a dynamic number of elements or if you don't know the size in advance.

2. No Built-in Bounds Checking: C arrays do not provide built-in bounds checking. Accessing an element outside the array's bounds can lead to undefined behavior, such as memory corruption or crashes. You need to manually ensure that you stay within the array's boundaries.

3. Inefficient Insertions and Deletions: Inserting or deleting elements within an array is relatively inefficient. When you insert or remove an element, you may need to shift other elements, which can be time-consuming in large arrays.

4. Wasted Memory: If you allocate a large array but only use a small portion of it, you are wasting memory. This can be especially problematic when memory is limited.
